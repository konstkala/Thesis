ΟΔΗΓΟΣ: ΠΩΣ ΝΑ ΚΆΝΕΤΕ ΈΝΑ WORD DOCUMENT ΑΠΟ ΤΟ THESIS_GREEK_DRAFT.txt
================================================================================

Έχετε τώρα το προχέρι της διπλωματικής εργασίας σας στα Ελληνικά. Ακολουθήστε 
αυτά τα βήματα για να το μετατρέψετε σε ένα επαγγελματικό Word document.

================================================================================
ΒΉΜΑ 1: ΑΝΤΙΓΡΑΦΉ ΣΤΟ WORD
================================================================================

1. Ανοίξτε το αρχείο THESIS_GREEK_DRAFT.txt (αυτό που έφτιαξα)
2. Επιλέξτε όλο το κείμενο (Ctrl + A)
3. Αντιγράψτε (Ctrl + C)
4. Ανοίξτε ένα νέο Word document
5. Επικολλήστε (Ctrl + V)
6. Αποθηκεύστε ως "Διπλωματική_Εργασία.docx"

================================================================================
ΒΉΜΑ 2: ΜΟΡΦΟΠΟΊΗΣΗ
================================================================================

Για κάθε τμήμα:

A. Τίτλος Κεφαλαίου
   - Μέγεθος: 18pt
   - Bold
   - Γραμματοσειρά: Times New Roman ή Calibri

B. Υποτίτλος (π.χ. 2.1 Τίτλος)
   - Μέγεθος: 14pt
   - Bold
   - Ίδια γραμματοσειρά

C. Κυρίως Κείμενο
   - Μέγεθος: 12pt
   - Απλή
   - Ίδια γραμματοσειρά

D. Πίνακες
   - Κέντρο
   - Διαχωρισμός σειρών με σκούρο φόντο για κεφαλίδα

E. Εξισώσεις
   - Χρησιμοποιήστε εξισώσεις Word (Insert > Equation)

================================================================================
ΒΉΜΑ 3: ΠΡΟΣΘΈΣΤΕ ΑΡΙΘΜΗΣΗ ΣΕΛΊΔΩΝ
================================================================================

1. Πατήστε Insert > Page Numbers
2. Επιλέξτε θέση (κάτω δεξιά - standard)
3. Ξεκινήστε από σελίδα 1 ή όπως θέλει η σχολή σας

================================================================================
ΒΉΜΑ 4: ΔΗΜΙΟΥΡΓΊΑ ΠΊΝΑΚΑ ΠΕΡΙΕΧΟΜΈΝΩΝ
================================================================================

1. Τοποθετήστε τον κέρσορα όπου θέλετε τον πίνακα (συνήθως αμέσως μετά τη 
   δεύτερη σελίδα)
2. Πατήστε References > Table of Contents
3. Επιλέξτε ένα στυλ
4. Το Word θα δημιουργήσει αυτόματα τον πίνακα αν έχετε χρησιμοποιήσει 
   σωστά τα headings

ΣΗΜΑΝΤΙΚΌ: Πρέπει να χρησιμοποιήσετε Styles για τις επικεφαλίδες (Heading 1, 
Heading 2, κ.λπ.) αντί να τις κάνετε bold χειροκίνητα.

================================================================================
ΒΉΜΑ 5: ΠΡΟΣΘΈΣΤΕ ΕΙΚΌΝΑ ΤΗΣ MAPPER VISUALISATION
================================================================================

1. Ανοίξτε το αρχείο ../data/mapper/mapper.html σε ένα web browser
2. Τραβήξτε ένα screenshot (Print Screen)
3. Ανοίξτε ένα image editor (Paint, Photoshop, κ.λπ.)
4. Επικολλήστε
5. Κόψτε στο κατάλληλο μέγεθος
6. Αποθηκεύστε ως PNG

Στο Word:
1. Πατήστε Insert > Pictures
2. Επιλέξτε την εικόνα
3. Δεξί κλικ > Wrap Text > Square
4. Μετακινήστε στο κατάλληλο σημείο

Καλό σημείο για την εικόνα: Μετά τη σχέση αποτελεσμάτων (Κεφάλαιο 5.2)

================================================================================
ΒΉΜΑ 6: ΠΡΟΣΘΈΣΤΕ ΠΊΝΑΚΕΣ
================================================================================

Ο κώδικας ήδη έχει πολλούς πίνακες ASCII. Για καλύτερη εμφάνιση:

1. Επιλέξτε κάθε πίνακα ASCII
2. Insert > Table
3. Καθορίστε αριθμό γραμμών και στηλών
4. Αντιγράψτε τα δεδομένα

Ή δημιουργήστε πίνακες από τα CSV αρχεία:

../results/node_analysis_detailed.csv
../results/subject_analysis.csv

Δοκιμάστε: Ανοίξτε τα CSV αρχεία σε Excel, αντιγράψτε, επικολλήστε στο Word.

================================================================================
ΒΉΜΑ 7: ΕΚΤΕΝΉΣ ΑΝΑΘΕΏΡΗΣΗ
================================================================================

Περιεχόμενο που χρειάζεται επέκταση:

1. Θεωρητικό Υπόβαθρο (Κεφάλαιο 3)
   Προσθέστε μαθηματικές εξηγήσεις για:
   - Περιοδική Ομολογία (Persistent Homology)
   - Simplicial Complexes
   - DBSCAN αλγόριθμος
   
   Πόσο: +5 σελίδες

2. Λεπτομερέστερη Ανάλυση Αποτελεσμάτων (Κεφάλαιο 5)
   Προσθέστε:
   - Γραφήματα κατανομής καθαρότητας
   - Θερμοχάρτες σύγχυσης
   - Ανάλυση ανά ζώνη εμπιστοσύνης
   
   Πόσο: +5 σελίδες

3. Σύγκριση με Υπάρχοντα Συστήματα (Κεφάλαιο 7.3)
   Επεκτείνετε με:
   - Αριθμητικές συγκρίσεις
   - Χαρακτηριστικά σύγκρισης
   - Μελέτες που χρησιμοποιούν τα συστήματα
   
   Πόσο: +3 σελίδες

4. Παράρτημα Α: Κώδικας Python
   Προσθέστε:
   - s0_prp_raw.py
   - s1_load_and_view.py
   - s2_preproc.py
   - s3_embeddings.py
   - s4_umap.py
   - s5_mapper.py
   - analyze_mapper_results.py
   
   Πόσο: +10 σελίδες

5. Παράρτημα Β: Δεδομένα
   Προσθέστε:
   - Πίνακες κόμβων (node_analysis_detailed.csv)
   - Ανάλυση ανά υποκείμενο
   
   Πόσο: +3 σελίδες

6. Περισσότερες Αναφορές
   Προσθέστε αναφορές από:
   - KeplerMapper papers
   - UMAP papers
   - Biometric security papers
   - Fingerprint databases
   
   Πόσο: +2 σελίδες

Σύνολο από ανωτέρω: +28 σελίδες
Τρέχον κείμενο: ~15 σελίδες
Σύνολο: ~43 σελίδες

Για να φτάσετε στις 100 σελίδες, προσθέστε:
- Πιο λεπτομερή θεωρητικό υπόβαθρο (State of the Art)
- Πλήρη αποτελέσματα για κάθε υποκείμενο
- Συγκριτικές μελέτες με άλλα συστήματα
- Λεπτομερές παράρτημα με κώδικα

ΣΥΝΟΠΤΙΚΟΣ ΈΛΕΓΧΟΣ ΣΧΟΛΏΝ
================================================================================

Πολλές σχολές έχουν ειδικές απαιτήσεις. Βεβαιωθείτε ότι:

□ Μέγεθος Σελίδας: A4 (standard)
□ Περιθώρια: 2.5cm αριστερά, 2.5cm δεξιά, 2.5cm πάνω, 2.5cm κάτω
□ Γραμματοσειρά: Times New Roman ή Calibri 12pt
□ Διάστιχο: 1.5 (ή 2.0 - ελέγξτε την πολιτική της σχολής)
□ Αριθμηση Σελίδων: Κάτω δεξιά, ξεκινώντας από την πρώτη σελίδα δεδομένων
□ Πίνακας Περιεχομένων: Υπάρχει
□ Περίληψη: Υπάρχει (στα Ελληνικά και Αγγλικά)
□ Αναφορές: Υπάρχουν και σωστά μορφοποιημένες
□ Παράρτημα: Εάν απαιτείται

ΣΥΜΒΟΥΛΈΣ ΣΥΓΓΡΑΦΉΣ
================================================================================

Γενικές συμβουλές για ακαδημαϊκή συγγραφή:

1. Χρήση Τρίτου Προσώπου
   Χρησιμοποιήστε "Το σύστημα δείχνει" αντί "Δείχνουμε"
   (αν και κάποια τμήματα μπορεί να δεν το απαιτούν)

2. Ενεργητική Φωνή
   Προτιμήστε "Εφαρμόσαμε τον αλγόριθμο" αντί "Ο αλγόριθμος εφαρμόστηκε"

3. Ακρίβεια
   Χρησιμοποιήστε ακριβείς αριθμούς και ποσοστά (όχι "περίπου", "φαίνεται κ.λπ.")

4. Δομή
   Κάθε παράγραφος θα πρέπει να έχει:
   - Θέση (τι λέγεται)
   - Σώμα (λεπτομέρειες)
   - Συμπέρασμα (σύνδεση με την επόμενη παράγραφο)

5. Αναφορές
   Παραπέμψτε σε όλα τα στοιχεία που δεν είναι των δικών σας
   π.χ. "Σύμφωνα με τον Singh et al. (2015)..."

ΠΡΑΓΜΑΤΙΚΌ ΠΑΡΆΔΕΙΓΜΑ: ΠΏΣ ΝΑ ΞΕΚΙΝΉΣΕΤΕ
================================================================================

1. Ανοίξτε ένα νέο Word document
2. Ορίστε τα περιθώρια (Layout > Margins > Normal)
3. Ορίστε τη γραμματοσειρά (Times New Roman 12pt)
4. Δημιουργήστε την πρώτη σελίδα:

---ΞΕΚΊΝΗΜΑ WORD---

                    ΔΙΠΛΩΜΑΤΙΚΉ ΕΡΓΑΣΊΑ
                    
    ΤΟΠΟΛΟΓΙΚΉ ΑΝΆΛΥΣΗ ΔΑΚΤΥΛΙΚΏΝ ΑΠΟΤΥΠΩΜΆΤΩΝ
         ΓΙΑ ΒΙΟΜΕΤΡΙΚΉ ΤΑΥΤΟΠΟΊΗΣΗ
         
    Φοιτητής: [Το όνομα σας]
    Επιβλέπων Καθηγητής: [Όνομα καθηγητή]
    
    Πανεπιστήμιο: [Όνομα πανεπιστημίου]
    Τμήμα: [Όνομα τμήματος]
    
    Ημερομηνία: Ιανουάριος 2026

---ΤΈΛΟΣ ΠΡΏΤΗΣ ΣΕΛΊΔΑΣ---

5. Προσθέστε νέα σελίδα (Ctrl + Return)
6. Προσθέστε περίληψη
7. Προσθέστε πίνακα περιεχομένων
8. Αρχίστε να συμπληρώνετε το κείμενο
