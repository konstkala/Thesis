SECTION 1: METHODOLOGY
--------------------------------------------------------------------------------

Our approach combines deep learning and topological data analysis:

1. Image Preprocessing:
   - CLAHE (Contrast Limited Adaptive Histogram Equalization)
   - Normalization to [0, 1] range for consistent input

2. Feature Extraction:
   - ResNet-18 pre-trained on ImageNet
   - 512-dimensional feature vectors from penultimate layer
   - Captures discriminative fingerprint patterns

3. Dimensionality Reduction:
   - UMAP (Uniform Manifold Approximation and Projection)
   - Projected to 2D for visualization
   - Preserves local and global topological structure

4. Topological Analysis:
   - KeplerMapper with UMAP as lens function
   - DBSCAN clustering within overlapping cubes
   - Configuration: n_cubes=10, overlap=30%, eps=0.5, min_samples=3

5. Security Metric:
   - Node Purity: proportion of most common subject in each node
   - Measures subject separability in topological space
